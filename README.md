Sales Dashboard - Power BI Project 📊
Overview

This project is a Dynamic Sales Dashboard created using Power BI to analyze sales performance and generate meaningful business insights through interactive visualizations.

The dashboard helps users monitor key metrics such as Sales Amount, Profit, Quantity Sold, Category Performance, Payment Modes, and State-wise Analysis.

Features 🚀
Interactive Quarter Filter
State-wise Sales Analysis
Monthly Profit Trends
Category & Sub-category Performance
Payment Mode Distribution
KPI Cards for:
Total Sales Amount
Total Quantity Sold
Total Profit
Clean and professional dashboard design
Tools Used 🛠️
Power BI Desktop
Data Cleaning & Transformation using Power Query
Data Modeling
DAX Measures
Data Visualization
Key Insights 📈
Clothing category recorded the highest quantity sold.
COD was the most used payment mode.
Profit varied significantly by month.
Tamil Nadu sales performance highlighted in state analysis.
